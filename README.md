# MOBA Dodge Game - Simple

A browser-based MOBA-style dodge game built with HTML5 Canvas.

## Play Now

[https://moba-dodge-simple.vercel.app/](https://moba-dodge-simple.vercel.app/)

## Development

### Running Locally

```bash
npm run dev
```

### Running Tests

This project includes a comprehensive automated test suite. See [TEST_README.md](./TEST_README.md) for detailed documentation.

```bash
# Run all tests
npm test

# Run tests with coverage
npm run test:coverage

# Run smoke tests (quick validation)
npm run test:fast
```

### In-Game Testing

When dev mode is enabled, you can:
- Press **F12** to open the test UI panel
- Press **~** or **`** to open the dev console
- Use commands like `test`, `testfast`, `testverbose` to run tests

See [TEST_README.md](./TEST_README.md) for complete testing documentation.
